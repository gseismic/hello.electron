# Electron 教程

这是一个 Electron 学习项目，通过循序渐进的方式帮助你掌握 Electron 开发。

## 目录

### [01_hello](./01_hello/) — 第一个 Electron 应用

最小示例，演示主进程、预加载脚本和 IPC 通信基础。

**核心知识点：**
- BrowserWindow 创建窗口
- 预加载脚本与 contextBridge
- IPC 主进程与渲染进程通信
- 安全最佳实践（contextIsolation、nodeIntegration）

---

## 开始学习

1. 进入 `01_hello` 目录
2. 运行 `npm install` 安装依赖
3. 运行 `npm start` 启动应用

## 环境要求

- Node.js 18+
- npm 9+
